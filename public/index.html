    createLogin utilise une méthode HTTP de type POST pour créer un nouvel utilisateur. Elle nécessite d'envoyer un objet JSON dans le corps (body) de la méthode, ex: {"nom":"TartampionJ", "motDePasse":"Emf123"}. D'autres champs peuvent être spécifiés: "domaine", "profil", "email", "initiales", "langue". Le retour est un objet "login" rempli ou avec le nom=null si la création n'a pas pu se faire (login existant).
