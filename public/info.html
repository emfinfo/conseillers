Cette route doit s'utiliser en POST ...

La route createLogin nécessite une méthode HTTP de type POST pour créer un nouveau compte d'utilisateur. Cela nécessite d'envoyer un objet JSON dans le corps (body) de la méthode. Exemple : {"nom": "TartampionJ", "motDePasse": "Emf123"}. D'autres champs peuvent être spécifiés : "domaine", "profil", "email", "initiales" et "langue". Si le compte utilisateur a pu être créé, le retour est un objet JSON rempli avec les propriétés du compte. Dans le cas contraire, des valeurs null seront retournées dans les propriétés de l'objet JSON.
